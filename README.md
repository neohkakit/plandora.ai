# Landing Page

A one-pager SaaS landing page built with Next.js and Tailwind CSS, optimized for [Vercel](https://vercel.com) deployment.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

1. Push this repo to GitHub.
2. Import the project at [vercel.com/new](https://vercel.com/new).
3. Vercel auto-detects Next.js — click **Deploy**.

Or use the CLI:

```bash
npx vercel
```

## Customize

- **Copy & branding** — edit `src/app/page.tsx` and `src/app/layout.tsx` metadata.
- **Colors** — tweak CSS variables in `src/app/globals.css`.
- **Product name** — search/replace `Launchpad` across the project.
